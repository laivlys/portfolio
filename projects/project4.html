Finding Whaley

Authors: Sylvia Lam

This interactive engagement project enables users to contribute whale sighting data by double-clicking on the map and filling out a pop-up form. Users can upload images, videos, audio clips, textual descriptions, along with the date and location of their sightings.

When a sighting is submitted, the data is stored securely in Firebase Storage and becomes visible on the map after a page refreshâ€”accessible to both the user and others. Each sighting is marked with a whale tail icon. Hovering over these icons reveals detailed information about the respective sightings.

For deeper exploration, users can filter the data by whale species or search by country to view sightings in specific regions. These features allow for customized and meaningful engagement with the data.
